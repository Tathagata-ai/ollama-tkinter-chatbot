# 🤖 Local AI Chatbot (Ollama + Tkinter)

A ChatGPT-like desktop AI assistant that runs completely offline using Ollama.

## 🚀 Features

* 💬 Real-time streaming responses
* 🧠 Conversation memory
* ⚡ Fast local inference (phi3 model)
* 🖥️ Clean Tkinter GUI
* 🔒 Fully offline (no API required)

## 🛠️ Tech Stack

* Python
* Tkinter
* Ollama
* Requests

## ▶️ How to Run

1. Install Ollama and pull model:

```
ollama pull phi3:mini
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
python app.py
```
